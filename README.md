# Neon Ark

**Advanced Breach Intelligence & Digital Footprint Scanner**

Neon Ark helps you discover where your personal data has been exposed in security breaches, assess risk with privacy scoring, and get remediation steps.

## Quick Start (Frontend + Backend)

```bash
# 1. Clone & install
git clone https://github.com/YOUR-USERNAME/neon-ark.git
cd neon-ark
pnpm install

# 2. Run development
pnpm --filter @workspace/breach-checker dev
